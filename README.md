# 坦克大战 🎮

经典坦克大战（Battle City）网页版，使用纯前端 HTML + CSS + JavaScript + Canvas 实现。

> **在线游玩**：[https://yzh11231.github.io/tank.list/](https://yzh11231.github.io/tank.list/)

---

## 游戏简介

消灭所有敌方坦克即可过关。每关 4 个敌人，全部击败后自动进入下一关。保护基地（⭐）不被摧毁，否则游戏结束。

## 游戏规则

- **目标**：每关击败 4 辆敌方坦克
- **生命**：初始 5 条命（♥♥♥♥♥），被击中后消耗一条并在安全位置复活
- **基地**：画面底部的 ⭐ 是基地，被摧毁则游戏失败
- **关卡**：消灭全部敌人后自动进入下一关，关卡越高地图越复杂
- **敌人种类**：
  - 🟤 **棕色坦克**（普通）：1 发击毁，基础分 100
  - 🔴 **红色坦克**（精英）：需要 2 发击毁，高难度关卡出现
- **地图元素**：
  - 🧱 **砖墙**：可被子弹击毁
  - ⬜ **钢墙**：子弹无法穿透
  - ⭐ **基地**：需要保护的核心目标
- **子弹**：击中砖墙会将其摧毁，击中钢墙则会消失

## 技术实现

### 技术栈

| 技术 | 用途 |
|------|------|
| **HTML5** | 页面结构与布局 |
| **CSS3** | 界面样式与动画 |
| **JavaScript (Canvas API)** | 游戏引擎与渲染 |

### 核心架构

整个游戏运行在一个 `<canvas>` 画布上，由以下模块组成：

```
gameLoop()
  ├── update()    ← 处理输入、更新坦克/子弹状态、碰撞检测
  └── draw()      ← 绘制地图、坦克、子弹、爆炸特效
```

**核心类：**

- **`Tank` — 坦克类**：管理移动、射击、碰撞检测、生命值。玩家和敌人共用，通过 isPlayer 参数区分
- **`Bullet` — 子弹类**：管理飞行、碰撞检测、砖墙摧毁逻辑
- **`Tank.canMove()` — 碰撞检测**：检查坦克四角所在的地图格子，确保不与墙壁或其他坦克重叠

**地图系统：**

地图为 26×26 的瓦片网格（每格 20px），瓦片类型包括 EMPTY、BRICK、STEEL、BASE。每关开始时通过伪随机算法生成砖墙布局，级别越高复杂度也越高。砖墙支持细粒度破坏（每块砖分为 4 个子块）。

**敌方 AI：**

敌人每隔一段时间随机改变行进方向，并有 30% 的概率朝玩家方向移动，行为有一定策略性而非完全随机。

## 本地运行

### 方式一：直接打开

```bash
git clone https://github.com/yzh11231/tank.list.git
cd tank.list
# 双击 index.html 即可在浏览器中运行
```

### 方式二：本地服务器（推荐）

```bash
# 使用 Python
python -m http.server 8080
# 浏览器打开 http://localhost:8080

# 或使用 Node.js
npx serve .
```
